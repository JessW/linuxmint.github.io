Mint Developer Center

This page is for developers interested in Mint-related projects including Cinnamon and other Mint tools.

  • The Projects page gives a list of projects maintained by Mint and short descriptions of each project.
  • The Contributing page provides some pointers to how to contribute to Mint projects.
  • The Documentation page hosts the documentation for Cinnamon, which may be of interest to Cinnamon extension developers. It includes tutorials and API documentation for different parts of Cinnamon.

The site adopts a minimalistic design because us developers cannot design better than this. Its source can be found on GitHub.
